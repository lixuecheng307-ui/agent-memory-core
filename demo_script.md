# 60 Second Demo Script

1. Initialize local SQLite memory.
2. Create `alpha_agent` and `job_agent` project memories.
3. Add one global memory.
4. Search with a synonym-style query.
5. Show project isolation.
6. Build context and show only allowed memories.
7. Disable one memory and show it disappears.
8. Delete one memory and show it disappears.
9. Show RetrievalTrace exclusions.
10. Run benchmark and open `benchmark_report.md`.

No cloud API key is required.
